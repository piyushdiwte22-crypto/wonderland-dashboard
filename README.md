# Wonderland RV + Outbound RVs — Marketing Dashboard

Monthly marketing report for director meetings: social growth (Vista Social) + CRM and email performance (ActiveCampaign) for Wonderland RV and Outbound RVs.

- **Live dashboard:** https://piyushdiwte22-crypto.github.io/wonderland-dashboard/
- **Data:** `data/report.json` — aggregate numbers only, no customer data. Updated monthly by a scheduled Claude task (1st of the month, 10am AEST).
- **Branding:** logos can be dropped into `assets/` as `wl-logo.png` / `ob-logo.png`.

No API keys or secrets live in this repo. The updater runs locally and commits only aggregated JSON.
